# Raytracing in a Weekend.

### Dependencies:

- C++ version 11
- CMake

### To run: 

- run the command 
				cmake . && make && ./raytrace > image.ppm
- use the following website to display the image. (If you do not have a ppm viewer installed.)
	https://www.cs.rhodes.edu/welshc/COMP141_F16/ppmReader.html
